# API Fetcher Project

Data fetcher using Python + Requests with automatic retries, rate-limit handling, and Streamlit UI. Handles API failures gracefully and visualizes JSON responses.

## Features
- Automatic retry with exponential backoff
- Handles rate limits and HTTP errors
- Interactive Streamlit dashboard

## Run locally

1. **Clone the repo**
   ```bash
   git clone https://github.com/Shivanshi9522/api-fetcher-project.git
   cd api-fetcher-project
   ```
2. **Install dependencies** 
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the app**
   ```bash
   streamlit run app.py
   ```
